## Installation
Once cloned run command "npm install" to install all dependencies.
use "mysql -u root -p"
lastly from the db folder use commands
* 'source db.sql'
* 'source schema.sql'
* 'source seeds.sql'

lastly run 'npm start' or  'node index.js' in order to start application
